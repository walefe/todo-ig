## Instalação

Instale my-project com npm

```bash
  git clone https://github.com/walefe/todo-ig
  cd todo-ig
  npm install todo-ig or yarn todo-id
  npm dev or yarn dev
```
    
## Stack utilizada

**Front-end:** React, vite, vitest, styled-components
